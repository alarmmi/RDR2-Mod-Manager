# RDR2 Mod Manager for Red Dead Redemption 2 PC Mods

**RDR2 Mod Manager** is an open-source project by **alarmmi** for organizing **Red Dead Redemption 2 mods on PC**. The goal is simple: make single-player RDR2 modding safer, cleaner, and easier to troubleshoot.

## What you can do with RDR2 Mod Manager

- Install and disable RDR2 mods without manually deleting files.
- Create profiles for graphics, realism, roleplay, testing, and vanilla-safe setups.
- Back up game files before changing your Red Dead Redemption 2 folder.
- Detect conflicting mod files before launch.
- Track LML mods, ASI plugins, ScriptHook-based files, Reshade presets, and loose files.
