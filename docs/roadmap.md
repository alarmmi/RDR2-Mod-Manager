# RDR2 Mod Manager Roadmap

This roadmap shows planned development phases for **RDR2 Mod Manager**.

## Phase 1: Repository foundation

- SEO-focused README.
- GitHub Pages documentation.
- TypeScript project skeleton.
- Example configuration and manifest files.
- Issue templates and GitHub Actions.

## Phase 2: CLI prototype

- Game path scanner.
- Manifest parser.
- Profile create/list/activate commands.
- Backup create/list/restore commands.
- Conflict detection report.

## Phase 3: Desktop UI

- Mod library view.
- Drag-and-drop import.
- Profile switcher.
- Backup manager.
- Conflict scanner screen.

## Phase 4: Safer automation

- Dry-run install plans.
- File verification.
- Hash cache.
- Recovery mode.
- Better logs.

## Phase 5: Community ecosystem

- Mod manifest examples.
- Translations.
- Compatibility database.
- Contributor docs.
- Release builds.

## Long-term vision

The goal is to become a trustworthy open-source Red Dead Redemption 2 mod manager focused on safe single-player mod workflows, strong backups, and clear troubleshooting.
