# RDR2 Modding Glossary

This glossary explains common **RDR2 modding** terms used in the documentation.

## ASI plugin

A plugin file ending in `.asi`. Many PC games use ASI plugins for modding workflows. Track these carefully because they often depend on loaders.

## Backup

A saved copy of files before a mod changes them. Backups help restore Red Dead Redemption 2 after a broken mod install.

## Conflict

A conflict happens when two mods try to change the same destination file or require incompatible versions of the same dependency.

## Game path

The folder where Red Dead Redemption 2 is installed.

## LML

A common shorthand for Lenny's Mod Loader, a community mod loader used by many Red Dead Redemption 2 mods.

## Manifest

A structured file that describes a mod, its files, dependencies, and conflicts.

## Mod library

The folder where RDR2 Mod Manager stores imported mods before applying them to a profile.

## Profile

A named mod loadout. Profiles let you switch between different Red Dead Redemption 2 mod setups.

## Reshade preset

A graphics preset that changes post-processing, colors, lighting, or effects through Reshade files.

## ScriptHook

A common term for script extension tools used by some single-player PC mods. Install dependencies only from trusted sources.
