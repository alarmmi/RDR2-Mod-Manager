# Security Policy

RDR2 Mod Manager is designed for safe single-player mod organization. Security issues matter because mod managers can move, copy, and restore local files.

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Report a vulnerability

Please open a private security advisory on GitHub or contact the maintainer through the repository profile.

Include:

- a clear description of the issue;
- steps to reproduce;
- affected operating system;
- affected RDR2 Mod Manager version;
- whether file deletion, path traversal, arbitrary execution, or privilege escalation is involved.

## Out of scope

The project does not support or accept reports about:

- bypassing Rockstar Launcher, DRM, or ownership checks;
- cheating, griefing, or multiplayer advantage;
- distributing copyrighted game files;
- stealing accounts, tokens, saves, or personal data.
