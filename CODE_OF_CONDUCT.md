# Code of Conduct

RDR2 Mod Manager should be welcoming to players, mod authors, translators, testers, and developers.

## Expected behavior

- Be respectful and constructive.
- Credit mod authors and community tools.
- Help beginners without gatekeeping.
- Keep discussions focused on safe single-player mod management.

## Unacceptable behavior

- Harassment, insults, or hate speech.
- Requests for piracy, DRM bypass, cheating, or malware.
- Sharing copyrighted game files.
- Publishing someone else’s private information.

## Enforcement

Maintainers may remove comments, close issues, or block users who violate this Code of Conduct.
