# Changelog

All notable changes to **RDR2 Mod Manager** will be documented in this file.

This project follows the spirit of semantic versioning: `MAJOR.MINOR.PATCH`.

## [0.1.0] - 2026-07-04

### Added

- Initial GitHub repository structure for RDR2 Mod Manager.
- SEO-focused README and documentation pages.
- GitHub Pages documentation configuration.
- TypeScript architecture skeleton.
- Example mod manifest and configuration files.
- Backup, compatibility, troubleshooting, and FAQ guides.
- Issue templates and GitHub Actions workflow.

### Security

- Added safe modding guidelines.
- Added policy against DRM bypass, cheating, and multiplayer abuse.
